## üîç Quick Reference

### Common Tasks

#### Setting Up Development Environment
```bash
# 1. Install dependencies
npm install

# 2. Set up environment variables
npm run setup:env

# 3. Generate Prisma client
npm run db:generate

# 4. Run migrations
npm run db:migrate

# 5. Seed database
npm run db:seed

# 6. Start dev server
npm run dev
```

#### Running Tests
```bash
# Run all tests
npm test

# Run specific test
npm test -- path/to/test

# Run with coverage
npm test -- --coverage
```

#### Deployment
```bash
# Check deployment readiness
npm run check:deployment

# Build for production
npm run build

# Start production server
npm start
```
